## Overview
This case study showcases a **Customer Insight and Profiling initiative**
for a global **automobile client**, aimed at enabling
**personalized customer communication** using **customer survey feedbacks**.

The solution applied **Natural Language Processing (NLP)** techniques to extract
sentiment, uncover key themes, and create actionable **customer personas**
that could be leveraged for future service and engagement recommendations.
