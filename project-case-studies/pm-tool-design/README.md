# Project Management Tool Design (Jira-like)

## Overview
This case study demonstrates how I approached the **design of a Jira‑like
project management tool**, focusing on **Agile delivery, PMO governance,
operational visibility, and scalability**.

The objective was not to build code, but to **design a production‑ready
system** that balances:
- Team autonomy
- Management visibility
- Enterprise governance
