# SAP MM DataHub for Demand & Supply Forecasting – CPG (Tobacco)

## Overview
This case study showcases the design and implementation of a **Supply Chain DataHub**
modeled on **SAP MM module data**, enabling improved **demand and supply forecasting**
for a **large CPG (tobacco) client in Europe**.

The solution consolidated multiple SAP MM tables into a unified analytical data layer
to support forecasting use cases, while ensuring **zero operational disruption**
through a **factory‑by‑factory rollout strategy**.
