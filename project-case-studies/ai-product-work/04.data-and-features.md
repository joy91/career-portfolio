# Data & Feature Design

## Data Sources
- Customer purchase history
- Product catalog and categories
- Transaction timestamps and values

---

## Key Features

### Behavioral Features
- User–item interaction matrix
- Purchase frequency
- Product co‑occurrence patterns

### RFM Metrics
- **Recency**: How recently a customer purchased
- **Frequency**: How often purchases occur
- **Monetary**: Total spend over time

RFM metrics were used to:
- Segment customers
- Weight recommendations
- Prioritize high‑value users
