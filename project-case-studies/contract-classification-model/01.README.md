## Overview
This case study showcases an **AI‑driven contract classification and information
extraction PoC** designed to help enterprises **analyze, validate, and govern
client contracts at scale**.

The solution uses **Natural Language Processing (NLP)** with an
**XGBoost‑based classification approach** to automatically identify and verify
critical contract components, improving efficiency, accuracy, and compliance.
