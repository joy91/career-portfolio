# Modeling Approach

## Why NLP + XGBoost?
- NLP enables understanding of unstructured legal text
- XGBoost provides strong performance on structured feature representations
- The approach balances accuracy, interpretability, and performance

---

## Model Overview
- NLP pipeline to extract textual features
- Supervised classification using XGBoost
- Separate classification logic for different contract components

---

## Design Philosophy
- Prefer robust, explainable models over black‑box complexity
- Ensure outputs can be validated by business users
