## Overview
This case study showcases the implementation of **Salesforce Customer Data Platform (CDP)**
and **Salesforce Marketing Cloud** for a **major transportation and ticketing client in Europe**.

The objective was to unify customer data, enable real‑time personalization,
and improve marketing effectiveness while ensuring **GDPR compliance**.
